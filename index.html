Desafio 4 - Simulador Interactivo

Funciones

Son un conjunto de instrucciones que se agrupan para realizar una tarea concreta. Se pueden ocupar una y otra vez, en cualquier parte del codigo. Se declaran iniciando con la palabra reservada function y luego se abren parentesis donde se puede o no colocar parametros que va a recibir la funcion, y luego se abren llaves, donde se escribe el bloque de codigo, es decir, las instrucciones que quiero que realice cuando llame a la funcion, cuando la ejecute.

Mi idea

Consiste en crear un simulador interactivo para cobrar las mesas de un Resto Bar, de acuerdo al consumo que tenga cada mesa habilitada.
